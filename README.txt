Dies ist meine Klonversion von Tetris, erstellt mit Python 3 und Pygame.
Es sollte ein funktionierendes und optisch einigermassen ansprechendes Spiel entstehen.
Beim Erstellen dieses Spieles hatte ich als Herausforderung auf eine Unterstützung weitgehend verzichtet,
daher ist meine Lösung teilweise sehr ineffizient.
Interessant und lehrreich war es, anschliessend die Herangehensweisen und Lösungen von anderen Programmierern zu analysieren.

This is my clone-version of tetris, made with python 3 and pygame.
Main goal was to create a functional and visually somehow appealing game.
While creating the game, i used as less help as possible to have a good challenge. 
Thus my programmed solution is functional but also very inefficient as well.
It shure was interesting to analize the solutions other programmers came up with afterwards.
